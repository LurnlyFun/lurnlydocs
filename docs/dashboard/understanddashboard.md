# ❓ What is the Dashboard?
![MnDDashboardMockup](/img/mobile-desktop-dashboard-mockup.png)

---

The Lurnly Dashboard serves as your central hub for managing study materials, tracking learning progress, and accessing quick actions. It is designed with a dual layout that automatically adjusts for desktop and mobile users.

It can be accessed by pressing "Flashcards" on the top menu, or by going to https://lurnly.fun/dashboard
