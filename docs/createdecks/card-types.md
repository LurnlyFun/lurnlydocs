# 📝 Card Types & Layouts

Lurnly allows you to create two main types of cards to vary your learning style. These can be selected within the card editor panel.

## 1. Normal Cards ❓
These are standard flashcards designed for testing your knowledge. You can choose from four different **Answer Modes**:

| Mode | Function |
| :--- | :--- |
| **🔄 Flip** | The classic flashcard. The answer is revealed when you tap or click the card. |
| **☑️ Choice** | Multiple-choice format. You can add up to 10 options and mark multiple correct answers. |
| **✍️ Input** | Requires you to type the answer exactly (case-insensitive) to succeed. |
| **🔢 Select** | A variation of multiple choice where only **one** answer can be correct. |

## 2. Informational Cards ℹ️
Informational cards are not quizzes. Instead, they act as study notes or transitions within your deck.
* Use these for definitions, reading passages, or introducing a new topic.
* Instead of a "Question," the main field is used for "Content."
* There is no "Correct Answer" required; the learner simply clicks **Continue**.
