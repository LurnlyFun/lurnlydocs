# Welcome to Lurnly Docs

Lurnly is an adaptive flashcard platform built for friction-less, private learning. **Everything lives in your browser** — no account, no cloud, you own your data completely.

## Key Features

| Feature | Details |
| :--- | :--- |
| **4 Answer Modes** | **Flip**, **Choice**, **Input**, or **Select** — test yourself the way that sticks |
| **Multimedia Cards** | Embed YouTube videos, audio clips, and images directly in any card |
| **Day Streak 👑** | Track study consistency — hit the crown milestone and keep your streak alive |
| **Zero-Cloud Privacy** | All data stays in your browser. Export to `.json` to back up or share with anyone |
