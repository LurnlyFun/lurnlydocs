# 📖 Komplektu pārvaldība

## Kas ir pēdējo komplektu sadaļa?
Pēdējo komplektu sadaļa nodrošina tūlītēju piekļuvi jūsu visaktīvākajiem materiāliem, sakārtotiem pēc pēdējo studiju vai izveides laika. Parādīto pēdējo komplektu skaits ir atkarīgs no ekrāna izmēra. Mobilajās ierīcēs tieši zem jaunākā komplekta sekundārie pēdējie komplekti tiek parādīti kompaktā režģī, lai ietaupītu ekrāna vietu.

**Katra komplekta karte satur ātras piekļuves izvēlni pārvaldībai:**

**▶️ Study:** Palaiž komplektu mācīšanās režīmā.

**✏️ Edit:** Atver komplektu izveides/rediģēšanas saskarnē.

**📤 Export:** Lejupielādē komplektu kā .json failu dublēšanai vai kopīgošanai.

**🗑️ Delete:** Neatgriezeniski noņem komplektu no jūsu lokālās krātuves.
