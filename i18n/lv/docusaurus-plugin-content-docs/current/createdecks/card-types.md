# 📝 Karšu veidi un izkārtojumi

Lurnly ļauj jums izveidot divus galvenos karšu veidus, lai dažādotu mācīšanās stilu. Tos var atlasīt karšu redaktora panelī.

## 1. Parastās kartes ❓
Tās ir standarta zibatmiņas kartes, kas paredzētas zināšanu pārbaudei. Varat izvēlēties no četriem dažādiem **atbilžu režīmiem**:

| Režīms | Funkcija |
| :--- | :--- |
| **🔄 Apvēršana** | Klasiskā zibatmiņas karte. Atbilde tiek atklāta, pieskároties vai noklikšķinot uz kartes. |
| **☑️ Izvēle** | Vairāku izvēļu formāts. Varat pievienot līdz 10 variantiem un atzīmēt vairākas pareizas atbildes. |
| **✍️ Ievade** | Pieprasa precīzi ierakstīt atbildi (bez reģistrjutīguma), lai gūtu panākumus. |
| **🔢 Atlase** | Vairāku izvēļu variācija, kurā var būt pareiza tikai **viena** atbilde. |

## 2. Informatīvās kartes ℹ️
Informatīvās kartes nav viktorīnas. Tās darbojas kā studiju piezīmes vai pārejas jūsu komplektā.
* Izmantojiet tās definīcijām, lasāmajiem fragmentiem vai jaunas tēmas ieviešanai.
* "Jautājuma" vietā galvenais lauks tiek izmantots "Saturam".
* Pareizā atbilde nav nepieciešama — studējošais vienkārši noklikšķina **Turpināt**.
