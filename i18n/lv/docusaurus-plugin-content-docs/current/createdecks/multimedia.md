# 🖼️ Multivides pievienošana

Jūs varat padarīt savas kartes saistošākas, pievienojot multividi. Katra karte (parastā vai informatīvā) atbalsta vienu multivides elementu, lai sniegtu vizuālu vai audiālu kontekstu.

## Atbalstītie multivides veidi
Sadaļā **Pievienot multividi** karšu redaktorā varat atlasīt vienu no šiem veidiem:

* **🖼️ Attēls**: Ielīmējiet attēla **URL**, lai tas tiktu parādīts virs jautājuma vai satura.
* **🎥 Video**: Ielīmējiet **YouTube URL**, lai iegultu video atskaņotāju tieši kartē.
* **🎵 Audio**: Ielīmējiet audio faila **URL**, lai iekļautu atskaņojamu skaņas klipu.

## 👁️ Tiešraides priekšskatījums
Pievienojot tekstu vai multividi, **tiešraides priekšskatījuma** panelis ekrāna labajā pusē nodrošina reāllaika skatu uz jūsu karti.
* Tas rāda tieši to, kā karte izskatīsies studējošajam.
* Tas atjauninās uzreiz, tiklīdz rakstāt vai maināt multivides URL.
* Ja laukus atstāsiet tukšus, tiks rādīts uzvednis "Sāciet rakstīt, lai redzētu priekšskatījumu."
