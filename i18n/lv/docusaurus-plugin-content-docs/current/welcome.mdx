# Laipni lūdzam Lurnly dokumentācijā

Lurnly ir adaptīva zibatmiņas karšu platforma, kas veidota nevainojamai, privātai mācīšanās pieredzei. **Viss atrodas jūsu pārlūkprogrammā** — bez konta, bez mākoņa, jūs pilnībā kontrolējat savus datus.
